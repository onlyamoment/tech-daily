⭐ 10.5kPython

自我进化 Agent：从 3.3K 行种子代码生长技能树，实现全系统控制，代币消耗减少 6 倍。

中文翻译 / 解读
GenericAgent：从小型种子代码长出 Agent 能力

GenericAgent 的卖点很直接：用较小规模的种子代码，逐步扩展出更完整的 Agent 能力。它关注的是 Agent 如何自我扩展、如何管理技能，以及怎样降低执行任务时的 token 消耗。

主要内容

项目把 Agent 能力看成可以生长的系统，而不是一次性写死的脚本集合。基础代码负责启动和组织，后续能力通过技能树逐步增加。这个思路和传统自动化工具不同，后者通常是人写好每个动作，Agent 只负责调用。

它还强调效率。Agent 项目常见的问题是 token 消耗很快失控，尤其是多轮规划、反思、工具调用叠在一起时。GenericAgent 把“怎么少花 token 做更多事”放到了重要位置。

为什么值得看

如果你关心的是 Agent 架构，而不是某个具体应用，这个项目值得研究。它提出的问题很实际：Agent 怎样积累技能？怎样复用经验？怎样避免每次任务都从零思考？

适合谁

适合做 Agent 框架、自动化系统、个人助手项目的开发者。普通用户可以先看概念，真正使用前还需要评估权限、安全和可控性。

GenericAgent：3K行代码的自进化AI Agent框架

GenericAgent 是一个极简的自进化自主 Agent 框架，核心代码只有约 3000 行。它让大语言模型（LLM）获得对本地计算机的系统级控制能力——包括浏览器、终端、文件系统、键鼠输入、屏幕视觉以及移动设备（通过 ADB）。整个 Agent Loop 更是只有约 100 行代码。

它的核心理念是：**不要预装技能，而是让技能自己生长出来。** 每次 Agent 解决新任务，都会自动把执行路径沉淀为一个可复用的 skill（技能模块）。使用越久，积累的技能越多，最终形成一个完全属于你自己的技能树。

这个项目本身的开发也完全由 GenericAgent 完成——从安装 Git、执行 `git init`，到每次的 commit message，全部自主完成。
主要内容自进化机制

GenericAgent 与其他 Agent 框架的核心区别在于它的自进化能力：

| 你的指令 | 首次执行 | 之后调用 |

|---------|---------|---------|

| "读取微信消息" | 安装依赖 → 逆向数据库 → 写读取脚本 → 保存为 skill | 一行命令 |

| "监控股票并提醒我" | 安装 mootdx → 构建筛选流程 → 配置定时任务 → 保存为 skill | 一行启动 |

| "用 Gmail 发送文件" | 配置 OAuth → 写发送脚本 → 保存为 skill | 直接可用 |

执行流程：[新任务] → [自主探索（安装依赖、写脚本、调试验证）] → [结晶执行路径为 skill] → [写入记忆层] → [下次相似任务直接调用]

核心特性
  • **自进化**：每次任务自动生成 skill，能力随使用增长
  • **极简架构**：核心代码约 3K 行，Agent Loop 仅 100 行左右，无复杂依赖
  • **强执行力**：注入真实浏览器（保留登录态），9 个原子工具直接控制系统
  • **高兼容性**：支持 Claude / Gemini / Kimi / MiniMax 等主流模型，跨平台
  • **Token 高效**：上下文窗口 <30K（其他 Agent 通常 200K-1M），层状记忆确保相关信息始终在范围内
记忆系统

五层结构让 Agent 在执行过程中持续积累经验：

  • **L0 — Meta Rules**：Agent 的核心行为规则和系统约束
  • **L1 — Insight Index**：快速路由和召回的最小记忆索引
  • **L2 — Global Facts**：长期运行积累的稳定知识
  • **L3 — Task Skills / SOPs**：完成特定任务的复用工作流
  • **L4 — Session Archive**：已完成会话中提炼的任务记录归档
界面与集成
  • **桌面 UI**：提供 Terminal UI（基于 Textual）和 Qt、Streamlit 界面
  • **Bot 接口**：支持 Telegram Bot，也支持 QQ / 飞书 / 企业微信 / 钉钉
  • **Codeg 集成**：可通过 Codeg 的统一界面同时调用 GenericAgent 和其他 Agent
快速上手

```bash

git clone https://github.com/lsdefine/GenericAgent.git

cd GenericAgent

pip install requests streamlit pywebview

cp mykey_template.py mykey.py

编辑 mykey.py 填入你的 LLM API Key

python launch.pyw

```

为什么值得看
  1. **代码量极小**：3K 行核心代码实现了完整的自进化 Agent，远低于 OpenClaw 的 53 万行和其他主流框架
  2. **真 Browser 控制**：注入真实浏览器，保留登录态，不是沙箱或 headless 模式
  3. **成本低**：30K 上下文窗口大幅降低 token 消耗，成功率反而更高
  4. **真正可用的自进化**：不是预装一堆插件，而是让能力在实际使用中生长
适合谁
  • 对 AI Agent 原理和实现感兴趣的开发者
  • 需要将 LLM 集成到本地自动化工作流的用户
  • 希望构建私有化、个性化 AI 助手的团队
  • 有能力维护 Python 环境的技术用户（项目需要配置 API Key）
